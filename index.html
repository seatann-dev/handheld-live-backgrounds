Handheld Live Backgrounds

Animated wallpapers for your Steam handheld's Home screen — drawn live behind your library, with tunable controls in the Quick Access Menu.

Decky Loader plugin Bazzite / SteamOS Game Mode Steam Deck · ROG Ally · Legion Go 3 themes
How to install — each theme is a Decky Loader plugin. You need Decky Loader installed on your handheld (Game Mode).
  1. Download a theme's Install zip below.
  2. Copy it to your handheld, unzip it into ~/homebrew/plugins/ (one folder per plugin).
  3. Restart Steam / reload Decky. Open the Quick Access Menu → the plugin → toggle it on and tune it.
The Source zip is the unminified code if you want to read or modify it. These run on the Bazzite/Linux Game-Mode side only, not Windows.
