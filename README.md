# Handheld Live Backgrounds

Animated live wallpapers for Steam handhelds (Steam Deck, ROG Ally, Legion Go), built as Decky Loader plugins. Each theme draws a live animated canvas behind your Steam Home, with tunable controls in the Quick Access Menu.

**Live site:** https://js-dev-dvm.github.io/handheld-live-backgrounds/

Browse the gallery, watch each theme animate live in your browser, then download the Install zip (and the Source zip if you want the code).
